## 题干

若 $$\lvert a \rvert \lt 1$$, $$\lvert b \rvert < 1$$, $$\lvert c \rvert < 1$$，证明：$$ ab + bc + ac > -1 $$

---

### 🏆 解法：构造对称乘积（仅需初中代数！）

我们通过一个巧妙的代数构造，**一步完成证明**。步骤如下：

---

#### **第 1 步：构造两个对称乘积**

考虑以下两个表达式：

$$
(1 + a)(1 + b)(1 + c) \quad \text{和} \quad (1 - a)(1 - b)(1 - c)
$$

---

#### **第 2 步：说明这两个乘积都大于 0**

因为 $$\lvert a \rvert < 1$$，所以：
- $$-1 < a < 1$$ ⇒ $$1 + a > 0$$ 且 $$1 - a > 0$$

同理，由 $$\lvert b \rvert < 1$$、$$\lvert c \rvert < 1$$ 可得：
- $$1 + b > 0$$，$$1 - b > 0$$
- $$1 + c > 0$$，$$1 - c > 0$$

因此：

$$
(1 + a)(1 + b)(1 + c) > 0$$, $$\quad (1 - a)(1 - b)(1 - c) > 0
$$

✅ 两个乘积都是**正数**！

---

#### **第 3 步：将两个乘积相加并展开**

计算它们的和：

$$
(1 + a)(1 + b)(1 + c) + (1 - a)(1 - b)(1 - c)
$$

分别展开（可用多项式乘法或分步展开）：

- $$(1 + a)(1 + b)(1 + c) = 1 + a + b + c + ab + bc + ca + abc$$
- $$(1 - a)(1 - b)(1 - c) = 1 - a - b - c + ab + bc + ca - abc$$

将两式相加，**奇次项抵消**：

$$
\begin{aligned}
&\big[1 + a + b + c + ab + bc + ca + abc\big] \\
+&\big[1 - a - b - c + ab + bc + ca - abc\big] \\
=&\ 2 + 2ab + 2bc + 2ca \\
=&\ 2\big(1 + ab + bc + ca\big)
\end{aligned}
$$

---

#### **第 4 步：利用“正数之和 > 0”推出结论**

由第 2 步，两个乘积都 $$> 0$$，所以它们的和也 $$> 0$$：

$$
(1 + a)(1 + b)(1 + c) + (1 - a)(1 - b)(1 - c) > 0
$$

结合第 3 步的结果：

$$
2\big(1 + ab + bc + ca\big) > 0
$$

两边同时除以 2（正数，不等号方向不变）：

$$
1 + ab + bc + ca > 0
$$

移项得：
$$
ab + bc + ca > -1
$$

✅ **证毕！**

---

### 🌟 为什么这个方法适合初中生？

- **仅用知识**：多项式展开、不等式基本性质、正负数判断；
- **无需**：极限、导数、分类讨论、高等代数；
- **逻辑清晰**：构造 → 展开 → 比较 → 结论；
- **技巧性强但易懂**：一旦看到构造，豁然开朗！
